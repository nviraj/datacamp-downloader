## Installation

```
pip install -e .

```

## Set Token

```
datacamp set-token <Token>
```

## List Courses

```
datacamp courses --refresh
```

## Download Options

All Courses

```
datacamp download --no-videos --subtitles none all
```

Specific Courses

```
datacamp download --no-videos --subtitles none <IDs separated by Space>
```
